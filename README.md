# Stillpoint Labs — Marketing Site

The parent-company marketing site for Stillpoint Labs. Astro static site deployed to Cloudflare Pages.

## Scripts

```
npm install      Install dependencies
npm run dev      Local dev server (http://localhost:4321)
npm run build    Production build → dist/
npm run preview  Preview the production build locally
npm run check    TypeScript + Astro type checking
```

## Deploy target

- Production: `https://stillpointlabs.dev` (custom domain, pending DNS wiring)
- Pages preview: `https://stillpointlabs-site.pages.dev`

## Stack

- [Astro](https://astro.build) — static site framework
- [Cloudflare Pages](https://pages.cloudflare.com) — hosting + CDN
- Cormorant Garamond + Inter — self-hosted from `public/fonts/`
