## Key Findings

1. **Atmosphere is a trust signal.** The purple-ombre background, the cricket sounds, the slow "take a deep breath" prompt — these are sensory promises about what the product is going to be. They set expectations and the product immediately delivers on them.
2. **Multi-select goal step.** Unlike Headspace's single-pick, Calm asks the user to select MULTIPLE goals (sleep, anxiety, focus, etc.). The implicit message: "we don't need to know exactly which one you are; tell us as many as fit."
3. **"7 Days of Calm" as a default introductory program.** Editorial default — the user doesn't have to pick where to start; the system has already picked.
