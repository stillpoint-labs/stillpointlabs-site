## Key Findings

1. **5 interests, not 20 questions.** Pinterest collapsed a longer interest-discovery flow into a 5-tap step ("Follow 5 topics. We'll build a custom home feed for you."). The header sets the deal explicitly: tiny ask, big payoff.
2. **The onboarding IS the product.** By selecting 5 interests, the user has already built the home feed they'll see next. There is no "onboarding mode" separate from "using the product."
3. **Locale-based pre-warming adds 5-10% activation across countries.** Free signals (locale, time-of-day, language) shape the candidate pool before the user makes any decision.
4. **Pinterest's mental contract is honest:** "you tell us a tiny bit; we infer the rest." It does NOT pretend to know more than the user told it. It does NOT pretend to be magic.
