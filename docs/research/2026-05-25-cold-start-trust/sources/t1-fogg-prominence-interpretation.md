## Key Findings

1. **Credibility = Prominence × Interpretation.** Both stages must fire. A trust signal that the user does not notice has zero effect. A trust signal the user notices but interprets negatively has *negative* effect. Designers tend to over-invest in trust signals nobody notices.
2. **User involvement modulates prominence.** Highly motivated users (researching a big purchase, evaluating health info) notice many more cues than satisficers doing quick triage. The Stillpoint Elisabeth-user has LOW involvement, so the trust cues she notices are a small subset of all cues you could ship.
3. **Topic and task modulate interpretation.** A user evaluating a finance app interprets "free" very differently than a user evaluating a meditation app interprets the same word.
4. **Experience modulates both.** Sophisticated users notice and discount marketing-language cues; novice users take them at face value.
