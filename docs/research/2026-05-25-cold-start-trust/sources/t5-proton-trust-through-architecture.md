## Key Findings

1. **"Trust must be earned through transparency and peer review."** Proton's stated framework. The trust mechanism is verifiability, not assertion.
2. **The aligned business model IS the trust argument.** Proton's pitch: "we make money from users paying us to protect them; we have no financial incentive to betray them." This is structural-trust (McKnight institution-based) in business-model clothes.
3. **Open source + independent audits = structural assurance.** A user who cannot inspect the code can trust that experts have inspected it.
4. **Named founders + transparency reports = competence + benevolence + problem-solving (Sirdeshmukh's three dimensions, all three).** Every published transparency report is a Sirdeshmukh problem-solving dimension event — "here's what happened, here's how we responded."
