## Key Findings

1. **YNAB's onboarding asks for substantial work up front: connect accounts, customize categories, assign every dollar.** The trust contract is: "you do real work in session 1; the system delivers durable value in session 30."
2. **The "trust the process" frame is YNAB's explicit trust-formation strategy.** Users are told the method works if followed, even if it feels uncomfortable. This is RULE-BASED trust (McKnight institutional category) applied at brand level.
3. **YNAB's higher initial dropout is the trade-off for longer-term retention.** YNAB user retention curves flatten higher than Mint's; the steep early curve filters for users who will become high-engagement long-term.
4. **"You don't need to think of every expense right away — add categories as they come up."** This is the closest YNAB gets to humble-onboarding. It's a small concession in an otherwise demanding flow.

## Inclusion Decision

**Decision:** Supporting (Rule 4 — 2 factors favor: relevance, contextual fit for Troth comparison)
